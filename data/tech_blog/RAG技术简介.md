# 检索增强生成（RAG）技术简介

RAG（Retrieval-Augmented Generation）是一种将信息检索与大语言模型生成能力相结合的技术范式。其核心思想是：在模型生成回答之前，先从外部知识库中检索与问题相关的文档片段，再将这些片段作为上下文传入模型，从而使模型能够生成更准确、更有依据的回答。

RAG 的标准流程分为三步：

1. **索引阶段**：将文档切分为较小的文本块（Chunk），通过 Embedding 模型转化为向量，存储到向量数据库中。
2. **检索阶段**：对用户输入的问题同样进行向量化，计算其与库中各文本块的相似度，召回 Top-K 最相关片段。
3. **生成阶段**：将召回的文本块拼接为 Prompt 的上下文部分，由大语言模型综合生成最终答案。

与直接使用大模型相比，RAG 有效降低了幻觉（Hallucination）发生的概率，同时支持知识库的动态更新，无需重新训练模型，在企业知识问答、文档助手等场景中得到广泛应用。
